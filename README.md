# Parrillada App

Aplicación web sencilla para gestionar y visualizar información relacionada con una parrillada.

## Estructura del proyecto
- **index.html**: Página principal de la aplicación.
- **styles.css**: Estilos y diseño visual.
- **script.js**: Lógica en JavaScript para la interacción.

## Características
- Interfaz web básica con HTML, CSS y JS.
- Separación clara entre contenido, estilos y lógica.
- Código fácil de mantener y escalar.
